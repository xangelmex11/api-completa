API Mercado

Bienvenido a la API del Mercado

Facilita la gestión de proveedores y productos mediante endpoints RESTful.

URL Base de la API

https://api-mercado-2.onrender.com

Utiliza esta URL base para acceder a los endpoints de la API.

Documentación de Endpoints

  • GET /proveedores
    Devuelve un proveedor aleatorio de la base de datos.
    Ejemplo de respuesta: 
    {
    "proveedor": {
        "ProveedorID": 1,
        "Nombre": "Proveedor Ejemplo",
        "Telefono": "123456789",
        "Correo": "ejemplo@proveedor.com",
        "Direccion": "Calle Ejemplo, Ciudad"
    }
}
  • GET /proveedores/buscar/<nombre>
    Busca proveedores que coincidan con el nombre proporcionado.
    Ejemplo de respuesta: 
    {
    "proveedores": [
        {
            "ProveedorID": 1,
            "Nombre": "Proveedor Ejemplo",
            "Telefono": "123456789",
            "Correo": "ejemplo@proveedor.com",
            "Direccion": "Calle Ejemplo, Ciudad"
        }
    ],
    "mensaje": "Proveedores encontrados"
}

  • GET /productos
    Devuelve un producto aleatorio de la base de datos.
    Ejemplo de respuesta: 
    {
    "producto": {
        "ProductoID": 10,
        "Nombre": "Manzanas",
        "Categoria": "Frutas",
        "Precio": 25.50,
        "UnidadMedida": "kg",
        "Cantidad": 100,
        "Disponible": true,
        "FechaAgregado": "2024-01-01",
        "Proveedor": "Proveedor Ejemplo"
    }
}
  • GET /productos/buscar/<nombre>
    Busca productos que coincidan con el nombre proporcionado.
    Ejemplo de respuesta: 
    {
    "productos": [
        {
            "ProductoID": 10,
            "Nombre": "Manzanas",
            "Categoria": "Frutas",
            "Precio": 25.50,
            "UnidadMedida": "kg",
            "Cantidad": 100,
            "Disponible": true,
            "FechaAgregado": "2024-01-01",
            "Proveedor": "Proveedor Ejemplo"
        }
    ],
    "mensaje": "Productos encontrados"
}
